# Live Class Commands

## Session 1: Setup
```bash
git clone <repo>
cd telco-churn-prediction
pip install -r requirements.txt
```

## Session 2: Docker
```bash
docker-compose build
docker-compose up -d
docker-compose ps
```

## Session 3: Kafka
```bash
# Start Kafka
docker-compose -f docker-compose.kafka.yml up -d

# Run producer
python kafka/producer_service.py

# Run consumer
python kafka/inference_service.py
```

## Session 4: Airflow
```bash
# Start Airflow
docker-compose -f docker-compose.airflow.yml up -d

# Access UI
http://localhost:8080
```

## Session 5: ECS Deployment
```bash
cd ecs-deployment
./run_all.sh
```
